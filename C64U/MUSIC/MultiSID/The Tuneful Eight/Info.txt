Hello!

The Tuneful Eight is a demo that was created for Evoke in 2019. It utilises eight SID chips.
See here: https://youtu.be/SwN7anqqDP8

Prior to loading the .prg file, the .cfg file should be loaded. This switches the settings such that 8 SIDs appear at the correct addresses in the memory map of the U64. In addition it sets the Audio Mixer correctly.

Have fun!

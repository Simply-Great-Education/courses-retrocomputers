Distributed with the Commodore 64 Ultimate with permission from DeeKay/Crest.


Quick steps:
Step 1: select "RunMeFirst.cfg" and select Load Settings, then select Ok.
Step 2: select "blu.reu" and load it into memory, then select Ok.
Step 3: select "BluREU_Crest.prg" and select Run.


This demo requires a 16MB REU.
